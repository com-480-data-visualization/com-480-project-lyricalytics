Understanding the emotion space

For each visualization, we will have an image-specific introduction for the final submission (milestone 3). As such, we include only a sketch of what a 2D-version might look like. For futher details, please check the project goal description document.

How do lyrics' emotions correlate with song features?

For our second visualization, we are looking at mean song features by emotion category. Are there clear observable differences between them? We will polish our current rough implementation to improve user friendliness.

How is each emotion reflected in lyrics' word choices?

Here, we have a word cloud for each emotion. Our main extra idea for it is making the visualization more interactive, starting with the most common words across all songs, and allow users to include or exclude emotion categories dynamically. Other ideas include: (1) enable comparison by displaying multiple word clouds side-by-side (e.g., joy vs. sadness); and (2) create word clouds based on noun-adjective pairs to better capture emotional nuances (as seen in Text Viz lecture, slide 22).

Do popular songs have distinct song features?

A scatter-enhanced violin plot capturing how song features relate to popularity. Here, we are presenting all features in a scrolling manner, but our final implementation will not have this behavior. We will either have more interactivity whereby we limit the number of plots on display and the user chooses which features to visualize, or, we will have a single unified view, which is our first "stretch" goal for this visualization.

Our extra ideas are: (1) having a single unified view, whereby we explore all features on a single plot, rather than our current 8 separate ones (this might require rethinking the y-axis, normalizing it to 1 similarly to the radar plot); (2) finding a better way to represent top-hits and non-top-hits to improve readability, maybe by replacing the violin plot by a full scatter plot (with some x-axis variance so a similar shape will be retained); and (3) exploring whether there are outlying emotions, and if so, visualize their feature correlation using a third color.

Understanding the emotion space

Our final visualization is a stacked bar chart visualization for genre and lyrical sentiments. It explores the distribution of emotions across genres, offering users an interactive and comparative view of sentiment trends.

Our extra ideas are: (1) improving genre selection with dropdown menus or grouped interfaces; (2) displaying the percentage of top hits below each bar, with a brief explanation; (3) switching the y-axis to percentages to address large count differences; (4) showing emotion names and counts within each bar layer for clarity (if the bar is too small, try to make it visible on hover).
